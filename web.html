ナンバープレート認識
と
その活用

本作品の方針

ナンバープレートをもっと活用したい

  1. ナンバープレート自体を認識するシステムは多いが、ナンバープレートの情報を認識して活用しているシステムの種類の幅が少ない。
  2. 情報を活用したものは主に工事現場や駐車場などでの車両管理に使われていて企業向けや官公庁向けのものが多い。

→民間向けや娯楽関係に活用してみる ＝くじ引きや自動音楽生成・再生、ナンバープレートの着せ替え加工など

認識システム

Google Cloud

  • Google Cloud Strage :
    画像データを保存する場所

  • Vision AI : Googleの画像認識サービス
    1. Object Localization : 物体検知をするもの
    2. Label Detection : 画像内の情報を自動で整理
    3. Text Detection : 文字を認識するもの

    活用システム一覧

    くじ引きシステム

    ナンバープレートの色や文字情報から車種を識別する。
    車種ごとにクーポンの内容が変わる。
    （例：普通自動車はお土産、貨物自動車は食品・飲料など）

    VisionAPIが取得した文字情報数字4桁をくじの抽選番号として活用する。
    当選したら自治体や駐車場のクーポンを発行する。

    地方振興や主要なパーキングエリア・サービスエリアの混雑回避、
    道路の渋滞緩和に繋がると考えています。

    楽曲作成システム

    ナンバープレートの色や数字を使って、音程やリズムを自動で調整していく。
    音楽についての知識や楽曲製作をすることに難がありました。
